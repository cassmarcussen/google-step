Week 4 Overview: Climate Reports

For this week, I chose to learn how to use the Google Charts and Google Maps APIs. Below, for the two Week 4 sections, I use these skills to represent data from data sets in an interactive, visual format. I chose to focus on climate change data sets for both of the components.

Week 4 Libraries: Google Charts

According to data.world (source linked below), the description of the data set is: Contributors evaluated tweets for belief in the existence of global warming or climate change. The possible answers were "Yes" if the tweet suggests global warming is occurring, "No" if the tweet suggests global warming is not occurring, and "I can't tell" if the tweet is ambiguous or unrelated to global warming. We also provide a confidence score for the classification of each tweet.
